Assignment 3.1 Deliverables: Stock Market Advisor Agent
=========================================================

Objective
---------
Build an AI agent that integrates an external stock market API for dynamic problem-solving. The agent accepts stock symbols, fetches real market data from Alpha Vantage, processes recent price and volume trends, and uses a language model to generate enriched buy, sell, or hold style recommendations.

Important Note
--------------
This project is for educational purposes only. The generated recommendation is not personal financial advice and should not be used as the only basis for investment decisions.

1. Python Script
----------------
The stock market advisor agent is implemented in the following files:

- src/stock_market_advisor/agent.py
- src/stock_market_advisor/main.py

agent.py contains the main StockMarketAdvisorAgent class. It loads API keys from the shared .env file, fetches Alpha Vantage daily stock data, extracts recent market signals, and sends a compact stock analysis to the language model.

main.py provides the command-line interface. It supports both single-symbol runs and multi-symbol runs, such as:

python -m stock_market_advisor.main --symbol AAPL,TSLA,MSFT

3. Reflection Report
--------------------
This agent demonstrates tool integration by connecting a language model to live stock market data. Instead of asking the model to rely only on general knowledge, the script first retrieves structured market information from Alpha Vantage. It then processes that raw data into a concise summary that includes the latest close price, daily close change, five-day close change, recent average volume, and latest volume compared with the recent average.

This tool integration enhanced the agent's capabilities because the model received current, numeric context before generating its response. The external API provided real-world data, while the language model converted that data into a readable explanation and recommendation. This combination made the output more grounded and useful than a generic stock market response.

The main implementation challenge was connecting the structured API response to a prompt that the language model could use effectively. Alpha Vantage returns nested JSON data, so the agent needed to extract the useful fields and avoid sending too much raw data to the model. Another challenge was handling operational issues such as API keys, possible rate limits, missing data, API response messages, and making sure the user sees clear errors when something goes wrong.

The agent also includes a multi-stock feature, allowing several symbols to be analyzed in a single run. This supports the bonus task by making it possible to compare outputs for multiple stocks without restarting the script each time.

4. Reflection Questions
-----------------------

Question 1: How did integrating the stock API improve the AI's real-world applicability?

Integrating the Alpha Vantage API made the agent more applicable to real-world use because it allowed the model to work with actual stock data rather than relying on static training knowledge. The agent could retrieve recent prices and volumes, calculate trends, and provide recommendations based on current market signals. This made the response more relevant, timely, and specific to the user's chosen stock symbol.

Question 2: What insights did the AI generate that were enhanced by the external tool?

The AI's insights were enhanced by data such as the latest close price, daily price movement, five-day trend, trading volume, and volume compared with the recent average. These values gave the model concrete evidence for its recommendation. For example, a recommendation to hold or be cautious could be linked to short-term price movement, trend direction, or unusually high or low volume.

Question 3: How could this agent be extended to include additional features like multi-day forecasts or news sentiment analysis?

The agent could be extended by fetching longer historical time series data and calculating technical indicators such as moving averages, volatility, or relative strength. It could also integrate a news API to collect recent headlines about each company and use sentiment analysis to understand whether current news is positive, negative, or neutral. Combining market trends with news sentiment would give the agent a broader basis for recommendations.

5. Bonus Feature
----------------
The agent supports multi-stock analysis in a single run. Users can enter comma-separated symbols, such as AAPL,TSLA,MSFT, and the agent will fetch, analyze, and generate recommendations for each symbol in sequence.
