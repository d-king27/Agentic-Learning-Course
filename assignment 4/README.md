# Assignment 4: Virtual Event Planning Agent

Objective: design and implement an AI agent that can plan and execute a
step-by-step strategy for organizing a virtual event.

The agent collects event details, generates a structured event plan, and lets
the user track progress by marking tasks as complete.

## Step 1: Set Up the Environment

Create and activate a virtual environment:

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
```

Install the required libraries:

```powershell
pip install -r requirements.txt
pip install -e .
```

Use the shared course-level `.env` file for your OpenAI key:

```text
Agentic-Learning-Course/.env
```

Example:

```text
OPEN_AI_KEY="your-openai-api-key-here"
OPENAI_MODEL="gpt-4"
```

## Step 2: Define the Workflow

```text
1. Collect event details:
   - event type
   - date
   - target audience
   - goals
2. Generate a detailed event plan.
3. Break the plan into phases and tasks.
4. Provide step-by-step guidance.
5. Track progress by allowing users to mark tasks as complete.
6. Show remaining tasks and completed tasks.
```

## Step 3: Run the Agent

Interactive mode:

```powershell
python -m virtual_event_planner.main
```

Single-run mode:

```powershell
python -m virtual_event_planner.main --type "Webinar" --date "December 15, 2024" --audience "Small business owners" --goals "Teach marketing strategies and generate leads"
```

## Step 4: Test the Agent

Use this sample input:

```text
Type: Webinar
Date: December 15, 2024
Audience: Small business owners
Goals: Teach marketing strategies and generate leads.
```

Review the generated event plan for clarity, completeness, and actionable
phases. In interactive mode, mark some tasks complete to simulate progress
tracking.
